Tag version: v1.1.1
Release title: v1.1.1 - 全新 UI、主题支持与体验优化

Description:
本次更新带来了全新的 MobaXterm 风格界面，并大幅优化了使用体验。

### 🎨 UI 重构与美化
- **全新视觉**：默认采用清爽的 **Sky Blue (MobaXterm 风格)** 主题，界面更现代、扁平。
- **主题切换**：新增**设置菜单**，支持一键切换 Sky Blue、Dark Mode (深色模式) 和经典浅色主题。
- **侧边栏升级**：左侧连接列表全面优化，增加了“用户会话”标题栏、连接线和层级图标，结构更清晰。
- **图标优化**：区分文件夹和远程桌面图标，直观易懂。

### ✨ 新增功能
- **标签页拖拽排序**：现在可以按住鼠标左键自由拖动标签页来调整顺序。
- **默认 CMD**：新建终端默认使用 CMD，更符合运维习惯。
- **右键菜单**：优化左侧列表交互，支持在空白处点击右键弹出菜单。

### ⚡ 体验优化
- **自动获取焦点**：
  - RDP 连接成功后，焦点会自动切换到远程桌面，无需手动点击。
  - 切换标签页时，焦点会自动切换到当前显示的远程桌面。
- **智能重连**：优化了自动重连机制，能正确识别 "被顶号" 的情况，避免无限重连。

### 🐛 问题修复
- 修复了拖拽标签页时可能导致的程序报错。
- 修复了点击标签页关闭按钮偶尔失效的问题。
- 修复了标签页文字垂直不居中和截断的问题。
